# THINKBOARD
Thinkboard is a modern, full-stack MERN application designed for seamless note-taking. It combines a sleek, responsive user interface with robust backend logic, featuring real-time rate limiting to ensure high availability and security.

# Key Features
- Full CRUD Operations: Create, read, update, and delete notes with ease.
- Intelligent Rate Limiting: Built-in protection against API abuse using Upstash Redis.
- Modern Design: A beautiful, responsive UI built with Tailwind CSS and DaisyUI.
- Real-time Feedback: Interactive toast notifications for user actions like saving or deleting.
- Dynamic UX: Smooth navigation and page transitions powered by React Router.

# Tech Stack
Frontend
- React 19: Modern UI library for component-based architecture.
- Vite: Lightning-fast build tool and development server.
- Tailwind CSS & DaisyUI: Utility-first styling with high-quality components.
- Lucide React: Clean and consistent iconography.

Backend
- Node.js & Express: Scalable server-side environment and routing.
- MongoDB & Mongoose: Flexible NoSQL database with schema modeling.
- Upstash Redis: Serverless Redis for global rate limiting.

# Getting Started
1. Prerequisites
- Node.js (v18+)
- MongoDB Instance (Atlas or Local)
- Upstash Redis Account

2. Installation
Clone the repository
git clone <https://github.com/Anoop840/Thinkboard.git>


Install Backend dependencies
- cd backend
- npm install

Install Frontend dependencies
- cd ../frontend/thinkboard
- npm install
